AR GEAR, Support Decentral AR World

AR GEARPlatform

ARGear Platform은 AR 서비스 및 생태계를 생산자 - 플랫폼 - 소비자 형태의 연결로 구성하고 ARGear World로 지칭한다.

  • ARGear Platform

    ARGear Application 구동 환경 및 마켓플레이스

  • AR Application Maker

    ARGear Platform 상의 어플리케이션 / 컨텐츠 생산자

  • AR Application Consumer

    ARGear Platform 상의 어플리케이션 / 컨텐츠 소비자

Application

ARGear Application Type

  • Native App (iOS / Android / Windows)
  • Web App (Chrome, FireFox, Safari, MS Edge 용 App)
  • Custom App (AR GLASS 등)
  • 단위 콘텐츠 (ASSET)

Engine

Core AR Engine

  • AR WebGL Engine

    WebGL기반 product 구동 엔진. AR Studio에서 제작된 바이너리 빌드를 실행 시키는 역할을 담당

  • ARGear Core Engine

    AR 서비스에서 기술적으로 단말에서 지원하는 AR관련 센서 및 하드웨어 인터페이스, 컴퓨터 비전, 이미지 프로세싱 기능들을 제공

  • Open API Add-on

    Core Engine 외에 외부에서 제작된 AR관련 엔진을 Open API Interface로 add-on 기능 제공

  • Open Authoring Tool Add-on

    기존 개발/저작 툴에서도 ARGear SDK을 사용할 수 있도록 플러그인 지원 예정

cloud

ARGear Application 및 관련 AR 컨텐츠들이 저장/유통/접속되는 공간

  • 모든 Asset들의 정보 접근 레이어
  • 계정별 접근 권한, 소유 권한, 소모 권한에 대한 정보 및 세션 관리
  • 각 Asset의 Hash 및 Cloud Link 정보 그리고 Asset에 대한 소모, 소유, 접근 권한에 대한 트랜젝션 관리
  • ARGear의 모든 데이터들이 Archive되는 Cloud상의 저장 공간
